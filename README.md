# 🌲 KR Timber Management System

A full-stack web application designed to manage timber inventory, sales, purchases, and expenditure tracking across multiple wood categories, enabling real-time stock management, accurate financial tracking, and centralized business analytics.
---

## 🚀 Features

* 🔐 User Authentication (Login & Signup)
* 📦 Inventory Management (Purchase & Sales)
* 📊 Real-time Stock Tracking
* 💰 Profit & Loss Calculation
* 🧾 Expenditure Management
* 📅 Date-based Filtering
* ✏️ Edit & Delete Transactions
* 📈 Monthly Grouping & Analytics
* 📱 Fully Responsive UI

---

## ⚙️ Installation & Setup

### 1. Clone the repository

git clone https://github.com/mddanish222/KR-Timber.git
cd KR-Timber

### 2. Setup Backend

cd backend
npm install

### 3. Create .env file

MONGO_URI=your_mongodb_connection_string

### 4. Run Server

node server.js
